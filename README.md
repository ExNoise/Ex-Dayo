# License

The TTS library used in "Ex-Dayo" is a reverse-engineered version of a Software called S.A.M. (Software Automatic Mouth) published in the 80s, is essentially an Abandonware, people reached out to the company who created it but the last life sign is from 2009 (the last website update).

I cannot put this code under any specific open source license, however it can be used as "Fair Use" as internation copyright law.
